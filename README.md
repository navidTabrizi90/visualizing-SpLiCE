# SpLiCE Implementation (Interpreting CLIP with Sparse Linear Concept Embeddings)

This repository contains a clean, modular implementation of the SpLiCE method:
- CLIP encoding
- Modality alignment
- Sparse non-negative decomposition
- Reconstruction of interpretable embeddings

Paper: Interpreting CLIP with Sparse Linear Concept Embeddings (NeurIPS 2024)
